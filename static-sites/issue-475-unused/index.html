Bench sequence

A patient path from damage to use.

The layout of every unused kit follows the same rhythm, so the second repair feels easier than the first.

  1. 01

    Identify the stress

    Match the crack, chip, or joint to a repair tray and read the use-after-repair caveat before opening compounds.

  2. 02

    Dry fit the object

    Use the printed bench map to test clamps, fingertip pressure, and sanding clearance while everything is still clean.

  3. 03

    Fill, set, and rest

    Apply only what the seam asks for. Cure times are written plainly, with kitchen-safe storage reminders.

  4. 04

    Return with limits

    Bring the piece back as a fruit bowl, desk cup, plant saucer, or shelf object when serving use is not advised.
