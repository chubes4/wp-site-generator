Fit-check pathway

Measure twice, order the shade once.

The path turns a lamp base and damaged shade into a confident cart: form, diameter, drop, fitter, harp height, panel material, lining, and trim width.

  1. Identify the socket. Note harp saddle, uno thread, or clip bulb.
  2. Measure the base. Match shade diameter to lamp height and visual weight.
  3. Set the drop. Keep socket hardware covered without crowding the bulb.
  4. Choose the finish. Compare swatches under the lamp's bulb temperature.
