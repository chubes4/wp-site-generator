The ledger method

A clean shelf is useful only when the numbering system survives the next firing.

01

Assign the tile before the glaze touches it.

Each tile ID connects to a batch card, recipe note, and firing-log row so there is no loose sample to interpret later.

02

Record variables where the work happens.

Waterproof cards sit beside the bucket for dip time, specific gravity, additions, and what the test is meant to prove.

03

Compare by clay body and cone.

Rails and tabs group results by the variables that change the outcome, not by the day someone cleaned the bench.

04

Archive what earns a repeat.

Successful surfaces move into a permanent binder and rail position with enough context to remake them next month.
