The kind of place you can understand from the sidewalk.

Juniper Table was built for ordinary neighborhood uncertainty: Is the kitchen open? Can I buy something good for dinner? Is there room for two? The answer should be visible before you open the door.

By day, the front shelves carry useful groceries, breads, jars, fruit, and coffee. By evening, the room shifts gently into shared suppers and small workshops, with the same counter, the same staff, and a menu stamped with the date.
